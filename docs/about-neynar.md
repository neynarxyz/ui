# About Neynar

**The easiest way to build on Farcaster.**

[Neynar](https://neynar.com) is the developer platform for Farcaster, providing the infrastructure, APIs, and tools that power the majority of apps in the ecosystem.

---

## What You Can Build

| Use Case | Description |
|----------|-------------|
| **Social Apps** | Connect Farcaster social graphs and user profiles to your applications |
| **Mini Apps** | Build and deploy interactive experiences that run natively in Warpcast |
| **AI Agents** | Deploy agents with contextual awareness and automated real-time interactions |
| **Clients** | Create apps with seamless onboarding, rich profiles, and scalable feeds |

---

## Platform Features

### APIs & Data
RESTful APIs for reading and writing to Farcaster—user profiles, casts, channels, reactions, feeds, and more. Real-time data streams and indexed databases for analytics.

### Managed Hubs
Run your own Farcaster hub without the operational overhead. We handle uptime, updates, and scaling.

### Webhooks
Real-time notifications for on-chain and off-chain events. Trade webhooks let you subscribe to Base transactions by FID—perfect for social trading, alerts, and copytrading tools.

### Neynar User Score
Reputation scoring used by developers across the ecosystem to manage spam and user quality in apps, agents, and products.

### Mini App Studio
Go from text prompt to deployed mini app. Farcaster-aware vibecoding with signed manifests and shareable URLs—no coding required.

### Notifications
22M+ notifications delivered. The best way to retain users in mini apps with batching and targeting capabilities.
